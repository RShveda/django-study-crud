Welcome to my demo project!

The purpose of this application was to practice CRUD operations by using Django generic CBVs.

Things I have learned:

  • Generic display views: DetailView and ListView.
  • Generic editing views: CreateView, UpdateView, DeleteView.
  • Customizing generic view methods: adding context data, customizing queryset, capturing self.kwargs from URLconf, etc.
  • Customizing model methods: save, delete.
  • Adding convenient navigation between pages using url template teg with args.

As a user you can access list of schools and see number of students by each of the schools. It is possible to create a new school and add new students to it. Also you can edit/delete students and schools, and re-assign students to a different school. Press button below to navigate to Schools page.
